# express-generator-TP

# Install
### Clone this repository or download it as an archive:
```
git clone https://github.com/julien-guillan/node-app
```
### Install dependencies
````
npm install
````
For debug logs:
```
$DEBUG="node-app:server" npm start
````

# Run
```
npm run start
```
